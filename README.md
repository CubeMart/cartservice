# CubeMart Cart Service

![CubeMart](https://img.shields.io/badge/CubeMart-Cart-0f766e)
![C%23](https://img.shields.io/badge/C%23-Service-512BD4)
![Redis](https://img.shields.io/badge/Redis-Backed-DC382D)

Shopping cart microservice for CubeMart with Redis-backed per-user cart storage.

## Responsibilities

- Add items to a user's cart
- Retrieve cart contents
- Empty carts after checkout
- Persist cart state in Redis
